title: Examples
---

To help you get started, we've provided example projects. They can all be found in the `/examples` folder on [Github](https://github.com/Jussinevavuori/kilpi/tree/main/examples). Find your example below.

<Aside type="warning">
  The examples are for Kilpi before version 0.20. Hold tight while we rewrite
  the examples for the new API!
</Aside>

- [**Next.js App Router — News Demo (RSC Version)**](https://github.com/Jussinevavuori/kilpi/tree/main/examples/next-rsc-demo)

  This project uses Next.js App Router & React Server Components for a simple news application to demonstrate implementing basic authorization concepts.

  Includes protected queries, protected actions, roles, server actions with `next-safe-action` and middleware, and more.

  Stack: [Better Auth](https://www.better-auth.com/), [Kilpi](https://kilpi.vercel.app), [Bun SQLite](https://bun.sh/docs/api/sqlite), [next-safe-action](https://next-safe-action.dev/), [Shadcn/UI](https://ui.shadcn.com/), [Tailwind CSS](https://tailwindcss.com/), [React Hook Form](https://react-hook-form.com/), [Zod](https://zod.dev).

- [**Next.js App Router — News Demo (Client version)**](https://github.com/Jussinevavuori/kilpi/tree/main/examples/next-client-demo)

  This project implements the same demo as the one listed above, but uses the `@kilpi/client` and `@kilpi/react-client` packages in places to demo protecting UI on the client.
