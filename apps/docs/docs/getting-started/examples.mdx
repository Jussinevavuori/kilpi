To help you get started, we've provided some example projects. They can all be found in the `/examples` folder on [Github](https://github.com/Jussinevavuori/kilpi/tree/main/examples). Find your example below.

### [**Next.js Articles App Demo**](https://github.com/Jussinevavuori/kilpi/tree/main/examples/next-rsc-demo)

> Simple Next.js CRUD app for articles to demonstrate basic Kilpi usage in a full-stack application, as well as usage on clients and in the UI, both on client and on the server.
>
> **Stack**: [Kilpi](https://kilpi.vercel.app), [React](https://react.dev/), [Next.js](https://nextjs.org/), [Better Auth](https://www.better-auth.com/), [Bun SQLite](https://bun.sh/docs/api/sqlite), [Shadcn/UI](https://ui.shadcn.com/), [Tailwind CSS](https://tailwindcss.com/), [React Hook Form](https://react-hook-form.com/), [Zod](https://zod.dev).
